Title : Satallite Image Classification

There are satallite images of Kolkata of 4 differnet bands RBand, GBand, BBand, IBand.
Each image of size 512 * 512.
The feature vector dimension is 4.

Classifer Used for Classification: Naive Bayes Classifier

Classes :- 1) River
           2) Non-River
           
Programming Environment : Jupyter Notebook

Programming Language : Python

Packages Used : 1) Numpy
                2) numpy.linalg
                3) matplotlib.pyplot
                4) matplotlib.image
                5) random

This folder contains "Satallite Image Analysis.ipynb" which can be execute by jupyter notebook.

The output folder contains the images generated after classification.

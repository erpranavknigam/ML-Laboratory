Title : Applying Simple Linear Regression on Self Prepared Dataset

Programming Environment : Jupyter Notebook

Programming Language : Python

Packages Used : 1) Numpy
                2) Pandas
                3) Matplotlib
        
This folder contains a file named "Linear Regression.ipynb" which can be executed by using Jupyter Notebook. For Execution open the file in Jupyter Notebook and go to Cell tab, inside Cell tab click on Run All. This will execute all the available cell sequentially.

The output folder contains two plots one is initial scatter plot and other is a scatter plot with fitting line.

Result : Coefficient of Determination: 0.9838278294506653
